# Spirit Animal Oracle 🔮🌙

An immersive AI-powered personality quiz that reveals a mystical spirit animal based on the user's choices using Google's Gemini API.  
The experience combines cinematic UI design, animated interactions, and generative AI storytelling to create a magical personality journey.

---

## ✨ Features

- 🌌 Animated fantasy-inspired UI
- 🧠 AI-generated spirit animal analysis
- 🔮 Dynamic poetic taglines & descriptions
- ⚡ Real-time Gemini API integration
- 📱 Fully responsive design
- 🌠 Interactive loading animations
- 🎭 Personality-based quiz system
- 🌲 Atmospheric visual effects

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript

### AI Integration
- Google Gemini API

### Deployment
- Vercel

---

## 🧠 How It Works

1. User answers 10 personality-based questions.
2. Answers are converted into a structured prompt.
3. The backend sends the prompt to Gemini API.
4. Gemini generates:
   - Spirit animal
   - Emoji
   - Poetic tagline
   - Personality description
5. The result is displayed with animated UI effects.
